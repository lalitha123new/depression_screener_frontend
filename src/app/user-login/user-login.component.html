Access to this app is currently through organisations based in India that request for an access for their members/staff.

You will need to enter the organisation code provided to your organisation in order to begin using this app.

Organizations can request access by sending a formal request at push.d.nimhans@gmail.com

If you do not have an organization code, you can access the same tool by registering as an individual at https://echargementalhealth.nimhans.ac.in/pushd/
